# DEFIASSETS
Google Script that returns list of all assets (tokens, pools, nft, claimable etc...)

### DIY:
- 1) Add up to 3 ERC, BEP20, MATIC addresses
- 2) Add specific protocols if needed
- 3) Press on Refresh
